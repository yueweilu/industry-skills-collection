# 行业技能图谱 (Industry Skills Collection) 📚

> [English](README.md) | [中文](README_zh-CN.md)

> 一个由社区驱动的各行业必备技能合集。

欢迎来到 **行业技能图谱**！这是一个开源项目，旨在建立一个涵盖不同职业所需技能的综合地图，从软件工程到数字营销、平面设计等等。

我们的目标是帮助初学者了解他们需要学习什么，并协助专业人士查漏补缺。

## 📂 浏览技能

按行业浏览技能：

- [💻 软件工程 (Software Engineering)](skills/zh-cn/software-engineering.md)
- *(更多行业即将推出！欢迎贡献！)*

## 🚀 如何使用

只需浏览 `skills/` 目录即可找到您感兴趣的行业。每个文件都包含该领域相关的分类技能、工具和概念列表。
