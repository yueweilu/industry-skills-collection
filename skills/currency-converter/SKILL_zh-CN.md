# 汇率转换技能 (Currency Converter)

> [English](SKILL.md)

使用 Frankfurter API（无需 API 密钥）转换不同币种之间的金额。

## 工具

### convert.py

**位置:** `skills/currency-converter/convert.py`

**用法:**
```bash
python skills/currency-converter/convert.py [金额] [源货币] [目标货币]
```

**示例:**
```bash
python skills/currency-converter/convert.py 100 USD CNY
```
