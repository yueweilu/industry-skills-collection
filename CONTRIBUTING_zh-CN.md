# 参与贡献 (Contributing)

> [English](CONTRIBUTING.md) | [中文](CONTRIBUTING_zh-CN.md)

感谢您有兴趣参与贡献！我们依靠社区的知识来使这个合集更加准确和全面。

## 如何贡献

### 1. 添加新行业

如果您发现缺少某个行业，请添加它！

1.  在 `skills/zh-cn/` 目录中创建一个新的 Markdown 文件。使用命名约定 `industry-name.md`（例如 `digital-marketing.md`）。
2.  如果可能，请同时在 `skills/en/` 中创建对应的英文版本。
3.  使用以下模板：

    ```markdown
    # [行业名称]

    ## 核心概念
    - 概念 1: 描述...
    - 概念 2: 描述...

    ## 工具与技术
    - 工具 A
    - 工具 B

    ## 软技能
    - 技能 X
    - 技能 Y
    ```

4.  在主 `README_zh-CN.md`（以及 `README.md`）中链接您的新文件。
5.  提交 Pull Request。

### 2. 更新现有技能

发现了错别字？缺少了关键工具？

1.  打开 `skills/zh-cn/` 或 `skills/en/` 中的相关文件。
2.  编辑内容。
3.  提交 Pull Request。

## 指南

- **保持清晰**: 使用简单的语言。
- **客观公正**: 专注于广泛认可的技能和工具。
- **分类整理**: 逻辑地分组技能（例如“初级”、“高级”或“工具”、“概念”）。

我们要期待您的贡献！
