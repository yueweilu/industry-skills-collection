# 参与贡献 (Contributing)

> [English](CONTRIBUTING.md)

很高兴您有兴趣参与贡献！本项目依赖社区的提交来构建一个丰富多样的 AI Agent 工具库。

## 🛠 如何添加新技能

要添加新技能，请按照以下步骤操作：

1.  **创建目录**:
    在 `skills/` 文件夹内，创建一个新的目录，使用短横线命名法（例如 `pdf-reader`, `currency-converter`）。

2.  **添加脚本**:
    编写一个 Python (或 Node.js) 脚本来执行任务。
    *   **要求**: 脚本必须将 **JSON** 格式的数据输出到标准输出 (stdout)。
    *   **要求**: 优雅地处理错误（返回包含 `error` 键的 JSON 对象）。
    *   **依赖**: 如果使用了外部库，请在 PR 描述中列出，我们会更新 `requirements.txt`。

3.  **创建文档**:
    在您的目录中创建一个 `SKILL.md` (英文) 文件。请参考现有技能的格式：
    *   **头部元数据 (Frontmatter)**: 包含 `name` 和 `description`。
    *   **工具 (Tools)**: 脚本路径和使用说明。
    *   **输入/输出 (IO)**: 示例输入和输出。
    *   **致谢 (Acknowledgments)**: 如果改编自其他项目，请在此注明。

4.  **添加中文文档**:
    创建一个 `SKILL_zh-CN.md` 文件，内容与英文版对应。

5.  **提交 Pull Request**:
    推送您的更改并开启 PR。我们会尽快审核！

## 🧪 测试

在提交之前，请使用 CLI 测试您的技能：

```bash
python main.py
```

确保您的技能出现在列表中并且可以正确执行。
